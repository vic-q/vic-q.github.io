hadoop环境搭建

Posted on 2018-08-12 | In 大数据

hadoop原生环境搭建

官方单节点搭建教程：https://hadoop.apache.org/docs/stable/hadoop-project-dist/hadoop-common/SingleCluster.html

下载hadoop，地址：http://www.apache.org/dyn/closer.cgi/hadoop/common/，本文以hadoop-2.8.3.tar.gz为例，解压缩，配置hadoop环境变量，指定haddop_conf_dir(hadoop-2.8.3/etc/hadoop)。

配置hdsf文件地址及节点复制策略，修改etc/hadoop/core-site.xml文件

修改etc/hadoop/hdfs-site.xml

测试本机登陆ssh，执行ssh localhost。如果不能登录，重新生产ssh秘钥，执行下面的脚本。

执行完尝试路径ssh locahost，这里可能会出现Permission denied (publickey)错误，更改文件权限，然后重启电脑。

ssh localhost登录成功后，执行$ bin/hdfs namenode -format初始化hdfs节点。

执行sbin/start-dfs.sh，启动hdfs。启动成功后访问http://localhost:50070/

创建hdfs目录执行 bin/hdfs dfs -mkdir /user

测试上传文件到hdfs目录，执行 bin/hdfs dfs -put etc/hadoop input

执行hadoop-mapreduce-examples，bin/hadoop jar share/hadoop/mapreduce/hadoop-mapreduce-examples-2.8.3.jar grep input output ‘dfs[a-z.]+’

从hdfs目录下载文件到本地磁盘，执行bin/hdfs dfs -get output output

完成后停止hdfs服务，sbin/stop-dfs.sh。

线段投影之和

Posted on 2018-08-08 | In 算法

题目

有n条平行于x轴的线段，求n条线段投影到x轴的距离之和。

解题思路

1、把n条线段按照x坐标大小排序。

2、第一条线段和第二条线段合并，将合并后的线段在和第n条线段合并，合并时需要判断线段相离，相交，重合的情况。

3、合并线段的时间复杂度为O(n)。
